📘 Formula for Senior 3

قوانين الصف الثالث الثانوي العام

ملخص شامل لجميع القوانين والمعادلات — كهربية، مغناطيسية، وفيزياء حديثة

⚡ القوانين الأساسية - الكهربية

شدة التيار والشحنة

I = Q / t
Q = N · e

فرق الجهد

V = W / Q

المقاومة الكهربية

R = V / I
R = ρ · L / A
📌 حيث: ρ = المقاومة النوعية (Ω·m)، L = الطول (m)، A = المساحة (m²)

القدرة الكهربائية

P = V · I = I² · R = V² / R
W = P · t = V · Q = I² · R · t

التوصيلية

σ = 1 / ρ = L / (R · A)

🔌 توصيل المقاومات وقوانين كيرشوف

التوصيل على التوالي

R_total = R₁ + R₂ + R₃ + ...
• التيار متساوٍ في جميع المقاومات • فرق الجهد الكلي = مجموع فروق الجهد

التوصيل على التوازي

1/R_total = 1/R₁ + 1/R₂ + 1/R₃ + ...
• فرق الجهد متساوٍ في جميع المقاومات • التيار الكلي = مجموع التيارات

قانون أوم للدائرة المغلقة

I = V_B / (R + r)
V = V_B - I·r
• V_B = القوة الدافعة للبطارية • r = المقاومة الداخلية • V = فرق الجهد على المقاومة الخارجية

قوانين كيرشوف

مجموع التيارات الداخلة = مجموع التيارات الخارجة
مجموع فروق الجهد = 0 (في أي حلقة مغلقة)

كفاءة البطارية

η = (V / V_B) × 100%

🧲 المجالات المغناطيسية

كثافة الفيض المغناطيسي

B = (μ · I) / (2π·d)
لسلك مستقيم طويل على بعد d منه

مركز ملف دائري

B = (μ · I · N) / (2r)

ملف لولبي طويل

B = μ · I · n
n = N / L
• n = عدد اللفات لكل وحدة طول • μ = معامل النفاذية المغناطيسية

الفيض المغناطيسي

Φ = B · A · cos(θ)

القوة المغناطيسية على سلك

F = B · I · L · sin(θ)
قاعدة فلمنج اليسرى

عزم الازدواج

τ = B · I · A · N · sin(θ)

🌀 الحث الكهرومغناطيسي

قانون فارادي

emf = -N · (ΔΦ / Δt)
emf = -N · (Δ(B·A) / Δt)

حث سلك متحرك

emf = B · L · v · sin(θ)

الحث المتبادل

emf₂ = -M · (ΔI₁ / Δt)
M = (μ · A₁ · N₁ · N₂) / L₁

الحث الذاتي

emf = -L · (ΔI / Δt)
L = (μ · A · N²) / l

التردد الزاوي

ω = 2π·f
T = 1 / f

📈 التيار المتردد

القيم العظمى والفعالة

emf(t) = N·B·A·ω·sin(ωt)
emf_max = N·B·A·ω
emf_eff = emf_max / √2 = 0.707 · emf_max

القدرة الكهربائية

P = emf_eff · I_eff = I_eff² · R

المحول المثالي

V_s / V_p = N_s / N_p
I_p / I_s = N_s / N_p
كفاءة المحول المثالي = 100%

كفاءة المحول غير المثالي

η = (V_s · I_s) / (V_p · I_p) × 100%
P_loss = I_eff² · R

🔄 دوائر التيار المتردد

المفاعلة الحثية

X_L = ω·L = 2π·f·L

المفاعلة السعوية

X_C = 1 / (ω·C) = 1 / (2π·f·C)

المقاومة الكلية (Impedance)

Z = √[R² + (X_L - X_C)²]

الرنين في دوائر RLC

عند الرنين: X_L = X_C
f_r = 1 / (2π√(L·C))
I_max = V / R

⚛️ الفيزياء الحديثة

طاقة الفوتون

E = h·ν = hc / λ

الظاهرة الكهروضوئية

h·ν = E_W + KE
h·ν = h·ν_c + (1/2)·m·v²

موجة دي برولي

λ = h / (m·v)

كمية الحركة للفوتون

p = h / λ = m·v

طاقة مستويات الهيدروجين

E_n = -13.6 / n² (eV)

الانتقال بين المستويات

ΔE = h·ν = hc / λ

نصف قطر بور

r = n·h / (2π·m·v)

معادلة أينشتاين

E = m·c²

📊 الثوابت الفيزيائية المهمة

الثابت الرمز القيمة الوحدة
شحنة الإلكترون e 1.6 × 10⁻¹⁹ C
ثابت بلانك h 6.625 × 10⁻³⁴ J·s
سرعة الضوء c 3 × 10⁸ m/s
كتلة الإلكترون m_e 9.1 × 10⁻³¹ kg
ثابت النفاذية (الفراغ) μ₀ 4π × 10⁻⁷ H/m
ثابت كولوم k 9 × 10⁹ N·m²/C²
وحدة الطاقة 1 eV 1.6 × 10⁻¹⁹ J
