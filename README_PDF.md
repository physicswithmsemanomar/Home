# قوانين الفيزياء - ملف PDF جاهز | Physics Laws - Ready PDF

## 🎯 للحصول على ملف PDF كامل بجميع القوانين | To get complete PDF with all laws:

### الطريقة السريعة | Quick Method:
1. افتح ملف `laws_page.html` في المتصفح
2. اضغط على زر **"طباعة القوانين كـ PDF"**
3. انتظر تحميل الملف تلقائياً

### مميزات PDF المُنشأ | Generated PDF Features:
- ✅ جميع قوانين الفيزياء للصف الثالث الثانوي
- ✅ خطوط واضحة تماماً (Times New Roman)
- ✅ تخطيط كتابي احترافي
- ✅ مناسب للطباعة والقراءة
- ✅ يعمل على الهواتف والحواسيب

### إذا لم يعمل الزر | If button doesn't work:
استخدم الطريقة اليدوية:
1. اضغط `Ctrl + P` (طباعة)
2. اختر "حفظ كـ PDF"
3. اضغط حفظ
